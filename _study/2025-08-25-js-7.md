# Iteration

```html
<script>
	var coworkers = ["james", "tim", "kevin"];
</script>
<ul>
  <script>
  	var i = 0;
    while (i < coworkers.length){
      document.write('<li>'+coworkers[i]+'</li>');
      i = i + 1;
    }
  </script>
</ul>
```
