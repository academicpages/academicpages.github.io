# Developer Tools

Mac Safari에서 개발자 도구를 사용하려면 Cmd+Opt+I 키를 누르면 된다. **Elements** 에서는 현재 웹 페이지의 HTML, JS 등 코드 구조를 살펴 볼 수 있고 이 상태에서 esc 키를 누르면 **Console** 이 함께 표시된다. 

**Console** 에서는 JS를 실행할 수 있는데, 현재 웹 페이지에 삽입 된 JS 처럼 동작하게 된다.
