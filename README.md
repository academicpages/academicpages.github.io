# harrygcoppock.github.io

Personal website for Harry Coppock, built with [Quarto](https://quarto.org/) and hosted on GitHub Pages.

## Sections

- **Overview** — Bio and background
- **AISI** — Role as Research Scientist at the UK AI Security Institute
- **10 Downing Street** — 10 Downing Street Innovation Fellow
- **Lectureship** — Visiting Lecturer at Imperial College London
- **AI for Health** — PhD research on bias in AI for Health

## Local development

1. [Install Quarto](https://quarto.org/docs/get-started/)
2. Run `quarto preview` to serve locally with live reload
3. Run `quarto render` to build the site

## Deployment

The site is deployed automatically via GitHub Pages using the workflow in `.github/workflows/publish.yml`.
