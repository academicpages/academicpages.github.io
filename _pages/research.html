Main Projects


1. Computational neurethological basis of primate grooming: this project combines neural modeling, imitation learning, and computer vision to develop a computational framework describing how primates use social touch to communicate.

2. Neural correlates of natural social primate behavior: this project combines wireless neurophysiology and computer vision to understand the neural processes across the macaque (social) behavioral repertoire. With: Camille Testard, Sebastien Tremblay

3. MacSuite: this project aims to deliver open-source computational tools to anyone studying primates within and beyond the lab.

Collaborations


1. Neonatologist gaze classification: this project aims to predict and quantify the attention/focus of a neonatologist during neonatal resuscitation. With: Heidi Herrick

2. Foraging and ADHD: this project explores the potential of an explore-exploit decision making task as a biomarker for ADHD diagnoses. With: David Barack, Vera Ludwig

3. So You Think AI Can Dance? This project aims to decompose myriad dance genre through novel self-supervised approaches. With: Ben Baker

4. Understanding transcendental phenomenology through immersive reality kinematics: this project combines virtual reality, computer vision, and philosophy to understand how we physically interact with novel environments. With: Scott Rennie, Razvan Sandru, Zachary Mainen
