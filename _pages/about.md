This project features a theme park control system that monitors the system computer's internal temperature (in Celcius), displays a webpage
for remote computer health monitoring, 
Theme parks have been booming in popularity - they offer a unique thrill and storytelling experience through the use of dark rides, drop towers, and rollercoasters. Since theme park rides utilize heavy and dangerous machinery, extensive real time control systems need to be implemented to ensure guests are as safe as possible while also protecting the ride and ride system. Without a real time
control system, vehicles could collide, computers could overheat, and guests could be harmed.

This system features:
a green LED to indicate the system is on and tasks are running concurrently
a red LED to indicate when an issue arrises (high internal computer temperature or emergency stop button pressed
an internal system computer temperature sensor to monitor system health and safety
an emergency stop button to ensure guest and equipment safety
a webserver to allow remote viewing of internal computer temperature stats (average and maximum values)
