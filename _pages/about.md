Theme parks have been booming as they offer thrill and unique storytelling through the use of rollercoasters,
drop towers, and dark rides.
Since theme park rides utilize heavy and dangerous machinery, extensive real time control systems need to be implemented.
These systems ensure the guests are as safe as possible while also protecting the ride and ride system.
Without a real timecontrol system, vehicles could collide, computers could overheat, and guests could be harmed.

This project in particular features:
a green LED to show system heartbeat and concurrency
a red LED to indicate an issue (high internal computer temperature or emergency stop button pressed)
an internal system computer temperature sensor to monitor system health
a webpage to remotely view computer temperature stats (average and maximum values recorded)
an emergency top button to suspend tasks for a set period to ensure guest and system safety
