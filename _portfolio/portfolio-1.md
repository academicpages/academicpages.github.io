## Project Overview

Streak Fit is an innovative virtual group fitness platform that combines real-time video conferencing with advanced machine learning-based pose detection. Users can join group fitness sessions and receive real-time feedback on their workout movements from an AI-powered pose detection model.

## Key Features

- **Video Conferencing Integration:** Real-time video streaming using WebRTC for seamless group interactions
- **AI Pose Detection:** TensorFlow-based pose detection model that monitors user movements in real-time
- **Personalized Feedback:** Instant corrective feedback and form analysis during workouts
- **Group Fitness Sessions:** Support for multiple participants in synchronized workout sessions
- **Performance Tracking:** Records workout history and progress metrics

## Technical Stack

- **Frontend:** React with WebRTC for video streaming
- **Backend:** Node.js with Sockets.IO for real-time communication
- **ML/AI:** TensorFlow and OpenCV for pose detection
- **Database:** MongoDB for user data and workout history

## Team & Impact

Led a team of 3 developers to design and implement the full platform. The project demonstrates how AI can enhance the fitness experience by providing professional-grade coaching in a virtual environment.
