AI-Mindcare is an AI dialogue product for psychological companionship scenarios, covering user management, multimodal conversation, emotion records, and psychological assessments.

Role: Backend Developer

Time: 2025.03 - 2026.03

Tech stack: Python, Flask, Redis, RAG, streaming responses

Key work:
- Introduced a RAG solution with psychology-related literature to improve answer professionalism.
- Optimized long-text generation from one-time return to streaming output, allowing users to see generated content in about 2-3 seconds.
- Used Redis to cache session context, login state, and frequently used assessment data to improve response stability.
- Improved multi-environment deployment and sensitive-data handling.
