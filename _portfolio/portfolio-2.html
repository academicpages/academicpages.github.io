Project Overview

Q&A Made Easy is a comprehensive Chrome extension designed to enhance question-and-answer sessions during online meetings and webinars. It leverages Natural Language Processing to automatically filter inappropriate and duplicate questions while providing an intuitive interface for moderators.

Key Features

  • NLP-Based Filtering: Automatically identifies and removes profane language and repetitive questions
  • Speech-to-Text Support: Users can submit questions by voice, making it accessible and convenient
  • Admin Dashboard: Real-time moderation interface for managing question flows
  • Question Ranking: Automatically prioritizes popular and relevant questions
  • Analytics: Provides insights on engagement metrics and popular topics

Technical Stack

  • Frontend: Chrome Extension API, React for the dashboard
  • NLP: Python-based NLP models for text filtering and classification
  • Backend: Node.js with Express for API endpoints
  • Database: MongoDB for storing questions and moderation logs
  • Speech Recognition: Web Speech API for voice input

Impact

This tool has been instrumental in improving the Q&A experience for virtual events, meetings, and webinars. It significantly reduces moderator workload by automating the filtering process, allowing them to focus on selecting the most impactful questions for discussion.
