## Project Overview

Travelo is a smart travel planning application that leverages Named Entity Recognition (NER) to automatically parse travel blogs and extract tourist attractions. Users can input travel blogs or travel guides, and the application intelligently identifies destinations and generates optimized travel itineraries.

## Key Features

- **Automatic Destination Extraction:** Uses NER to identify tourist attractions from unstructured text
- **Intelligent Itinerary Generation:** Automatically creates organized travel plans based on extracted locations
- **Route Optimization:** Plots efficient routes connecting destinations using map visualization
- **Blog Parsing:** Supports multiple text formats and can process lengthy travel documents
- **User-Friendly Interface:** Intuitive design for easy trip planning and modification

## Technical Stack

- **Frontend:** React with interactive map visualization
- **Backend:** Django REST framework for API endpoints
- **NLP:** BERT-based Named Entity Recognition model for destination extraction
- **Mapping:** MapBox or Google Maps API for route visualization
- **Database:** PostgreSQL for storing itineraries and user data
- **Deployment:** Docker containerization for easy scaling

## How It Works

1. User inputs a travel blog or description
2. NER model identifies and extracts all mentioned tourist destinations
3. Application categorizes destinations by location and type
4. Algorithm generates optimal travel route minimizing distance
5. User can view itinerary on interactive map and make modifications

## Use Cases

- **Travel Bloggers:** Quickly extract journey planning from travel narratives
- **Tour Operators:** Automate itinerary creation from client descriptions
- **Adventure Planning:** Intelligently organize multi-destination trips
- **Travel Research:** Identify popular tourist destinations from travel literature

## Impact

Travelo demonstrates the practical application of NLP in travel technology, significantly reducing the time required to plan multi-destination trips while ensuring optimal routing and comprehensive destination coverage.
